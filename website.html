FindMyPet IOS App

I first started on this project because I wanted to make an app that would provide something truly useful users.
One of the first thing that I thought of is when my Cat went missing (who you can see in the GIF)

It allows for a pet owner to create a pin with information about a lost pet, including a photo, contact information, and a description of the pet. Neighbors can also check the map near them if they see a lost pet nearby and contact the owner

This app was developed through Swift 5 and Xcode along with Apple's MapKit feature

Overall I learned a ton of invaluable knowledge in making this app. It took me a lot of time since I had never used Swift/Xcode before, but I'm happy with how it turned out.
